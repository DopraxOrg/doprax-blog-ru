Как поднять свой VPN на VPS за 5 минут — с V2Ray и Doprax  

Вы хотите настоящий, полностью контролируемый VPN — без посредников, без компромиссов в приватности и производительности. Не «облачный прокси», не «подписка на сервис», а ваш собственный сервер с V2Ray, развернутый за минуты — без ручной настройки, без знания systemd или Nginx.  

Вот как это работает на Doprax:  

1. **Зайдите в личный кабинет**  
   У вас уже есть аккаунт? Отлично. Если нет — регистрация занимает 20 секунд (поддержка криптооплаты, email или Telegram-авторизация).  

2. **Выберите ProVM и разверните сервер**  
   В дашборде нажмите «Deploy» → выберите любой регион (Hetzner, Vultr, DigitalOcean и др.) → укажите конфигурацию (от 1 vCPU / 1 ГБ RAM — хватит для V2Ray даже с десятками подключений) → нажмите «Deploy».  
   Сервер запускается за ~60 секунд. Вы получаете root-доступ, IP и SSH-ключ — всё готово.  

3. **Установите V2Ray одним скриптом**  
   Подключитесь через терминал:  
   ```bash
   ssh root@ваш-ip  
   ```  
   Запустите официальный скрипт установки:  
   ```bash
   bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)  
   ```  
   Он сам скачает бинарник, настроит службу и запустит V2Ray.  

4. **Настройте конфигурацию (минимум действий)**  
   Отредактируйте `/usr/local/etc/v2ray/config.json`:  
   ```json
   {
     "inbounds": [{
       "port": 443,
       "protocol": "vmess",
       "settings": {
         "clients": [{
           "id": "ваш-uuid-здесь",
           "alterId": 0
         }]
       },
       "streamSettings": {
         "network": "tcp"
       }
     }],
     "outbounds": [{"protocol": "freedom"}]
   }
   ```  
   Перезапустите: `systemctl restart v2ray`.  

5. **Подключитесь с Android (или любого устройства)**  
   Установите приложение *V2RayNG* → создайте новый профиль → введите:  
   - Адрес: ваш IP  
   - Порт: 443  
   - ID: тот же UUID из конфига  
   - Шифрование: *auto*  
   → Нажмите «Подключиться».  

Готово. Ваш личный, шифрованный туннель в интернет — без фильтрации, без логов, без ограничений.
