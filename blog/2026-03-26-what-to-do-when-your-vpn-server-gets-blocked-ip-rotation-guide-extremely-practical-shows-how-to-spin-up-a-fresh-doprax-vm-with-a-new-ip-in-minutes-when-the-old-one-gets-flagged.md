# Что делать, если ваш VPN-сервер заблокировали: как за минуты запустить новый IP на Doprax

Иногда IP-адрес вашего VPN-сервера попадает в чёрные списки — и соединение перестаёт работать. Это не сбой. Это часть нормальной эксплуатации инфраструктуры для интернет-свободы. Главное — не терять время на диагностику или ручную перенастройку. С Doprax вы получаете новый рабочий IP *за 90 секунд*. Без перезагрузки, без конфигурации, без остановки сервисов.

Вот как это работает — пошагово и без лишних слов.

---

## Шаг 1. Остановите старый сервер (опционально)

Если текущий сервер уже недоступен или вы хотите избежать конфликтов — остановите его через [Doprax Dashboard](https://app.doprax.com).  
👉 Перейдите в **«My Servers»**, выберите нужный ProVM → нажмите **«Stop»**.  
Это мгновенно освобождает IP и прекращает начисление часов.

> ⚠️ Примечание: остановка не удаляет сервер. Все данные, конфиги и диски сохраняются. Вы можете возобновить его позже — но для смены IP это не нужно.

---

## Шаг 2. Запустите новый сервер с новым IP — за 2 клика

1. В том же разделе **«My Servers»** нажмите **«Deploy New Server»**.  
2. Выберите любой доступный регион и провайдера (Hetzner, DigitalOcean, Vultr и др.) — каждый из них выдаёт *уникальный публичный IPv4*.  
3. Укажите минимальные параметры: `1 vCPU / 1 GB RAM / 25 GB SSD` — этого достаточно для V2Ray, Trojan или Xray.  
4. Нажмите **«Deploy»**.

✅ Через 60–90 секунд сервер будет готов.  
✅ У него будет *совершенно новый публичный IP*, не связанный с предыдущим.  
✅ Вы получите уведомление и прямую ссылку на консоль.

---

## Шаг 3. Разверните свой VPN за 1 командой (без копирования конфигов)

Используйте **Doprax App Market** — там уже есть готовые шаблоны:

- [`v2ray-core`](https://app.doprax.com/market/v2ray-core)  
- [`xray-core`](https://app.doprax.com/market/xray-core)  
- [`trojan-go`](https://app.doprax.com/market/trojan-go)

Выберите нужный → нажмите **«Install»** → подтвердите развертывание.  
Сервер автоматически скачает бинарник, применит ваш прошлый конфиг (если он сохранён в Environment Variables), запустит службу и откроет порты.

> 💡 Совет: сохраните `config.json` в **Environment Variables tab** своего сервера — тогда при каждом новом разворачивании он подтянется автоматически.

## Почему это работает — и почему другие платформы не могут повторить

| Фактор | Doprax | Обычный VPS / Облако |
|--------|--------|----------------------|
| **Новый IP за <2 мин** | Да — каждый деплой = новый IPv4 от другого пула провайдера | Нет — часто требует ручного запроса, задержек, оплаты доп. IP |
| **Автоматический перенос конфигов** | Да — через Environment Variables и App Market шаблоны | Нет — ручной SCP, редактирование файлов, проверка прав |
| **Оплата только за активное время** | Да — часовая тарификация, остановка = 0 ₽/₽/€ | Часто фиксированная ежемесячная плата, даже если сервер выключен |
